Block Elements D:

Tag Name Purpose
h1–h6 headings includes a heading (levels 1–6)
p paragraph includes a paragraph of text
table table includes a table
tr table row includes a row of data
th table header make a table header
td table data include a data table cell
ol ordered list makes a list with numbers
ul unordered list makes a list without numbers
li list item adds an item on the list
blockquote block quotation shows a formatted quotation
br break adds a line break
div divide "seperates" a block level element to be styled
